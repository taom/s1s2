# S1S2 Feature Entitlement Matrix

## Rules

- health logging, viewing, export, concerning trend alerts, guided breathing, and basic creature discovery are never paywalled
- premium marketing may mention only features that are live in the current phase
- features marked `coming later` may appear in roadmap messaging, but never as active premium benefits
- Common and Uncommon creatures are the long-term free-tier backbone

## Status Key

- `Free live` - available to all users now
- `Premium live` - available now to premium users only
- `Coming later` - planned but not marketable as live
- `Not in scope` - not part of the current phase

## Premium v1 Packaging Rule

Premium v1 should describe only these live benefits:

- full graph galaxy
- all live creature tiers that are actually shipped
- unlimited history
- advanced music modes
- music export
- ship customization if implemented

Do not list doctor reports or third-party wearable sync in premium marketing unless the feature is already live.
